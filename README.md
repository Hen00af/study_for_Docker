このリポジトリはDockerに関する勉強を記録したものである。
対外へ自分の勉強履歴を残すという寄りは自分のために雑多に勉強するためのリポジトリのため、このリポジトリを読んでもあまり勉強にならないかもしれません。
パブリックにしている理由は承認欲求です。

目標：　git clone <this repo> && docker compose up --detach --build　を実行するとdockerが起動し、自分の想定するアプリケーションが起動すること。
